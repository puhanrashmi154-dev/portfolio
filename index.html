01 / About

Understanding systems by breaking them

Background, mindset, and the way I approach security work.

I'm Rashmiranjan Puhan, a Computer Science undergraduate with a strong pull toward Cybersecurity, Network Security, and Security Operations. I'm drawn to the layer most people don't see — how packets actually move, where a handshake can go wrong, and what a request looks like right before it becomes an exploit.

Most of what I know comes from hands-on practice rather than theory alone: running scans in lab environments, tracing TCP sessions in Wireshark, intercepting requests in Burp Suite, and learning the OWASP Top 10 by trying to break things safely before trying to defend them.

I also build — responsive web applications with HTML, CSS, and JavaScript — because understanding how something is built makes it easier to understand how it can be attacked, and how to defend it properly.
